Rendering Pipeline

From Markdown source to paginated A4 preview

1
Trigger
User types in CodeMirror → cm.on("change") fires → debounce 800ms → triggerRender()
app.js editor.js
2
Split & Hash
Markdown is split into sections at each # H1 heading.
Each section gets a content hash. If hash unchanged → skip (incremental rendering).
Sec 0: Cover + Sommaire Sec 1: # Intro Sec 2: # Approach Sec N: ...
render.js splitMarkdownSections()
3
Markdown → HTML
Each changed section is parsed by marked with custom renderers. Frontmatter is extracted first. Custom extensions handle heading numbering, underlines, section colors, and mermaid diagrams.
Raw MD → Frontmatter → marked parse → French typo → Heading nums → HTML
render.js parseSectionMarkdownSync() marked extensions
4
Iframe Injection (Double-buffered)
Section HTML is wrapped in a full document (CSS + fonts + Paged.js + section-init.js) and loaded into a dedicated <iframe> via blob URL.
New iframe renders offscreen, then swaps with the old one — no flicker.

Each iframe contains

Complete HTML doc • pdf.css + paged.css • Google Fonts • Paged.js polyfill • section-init.js • Section content

Section 0 (Cover)

Hero page from frontmatter (title, doctype, acheteur, ref) • Auto-generated sommaire from all H1 headings

render.js wrapInDocument() buildCoverContentHtml()
5
Paged.js Pagination
Inside each iframe, section-init.js runs:
• Replace \newpage with page breaks
• Wrap headings with next sibling to prevent orphans
• Run Paged.js → breaks HTML into A4 pages with margins, headers, footers
• postMessage("section-ready") to parent with page count
section-init.js paged.polyfill.js
6
Swap & Scale
On section-ready message: pending iframe replaces old one. Frame is scaled (transform: scale()) to fit preview pane width. Page numbers are corrected cumulatively across all sections.
render.js scaleSection() updatePageNumbers()
7
Post-render Sync
Anchor map rebuilt (editor line numbers ↔ preview Y positions). Bidirectional scroll sync and click-to-navigate re-attached. Table widgets refreshed.
rebuildAnchorMap() setupScrollSync() setupPreviewClick() refreshTableWidgets()
sync.js app.js table-widget.js
