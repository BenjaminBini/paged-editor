External Files (fetched at startup)

  • css/pdf.css — PDF print stylesheet (fetched as text, injected as <style>)
  • css/paged.css — Paged.js viewer styles (fetched as text)
  • js/section-init.js — iframe init script (fetched as blob URL)
  • assets/paged.polyfill.js — Paged.js 900KB (fetched as blob URL)
  • assets/default.md — fallback document (fetched as text)
  • assets/beorn-logo.png — logo (converted to data URI)

Lazy-loaded (on demand)

  • mermaid@11 CDN — ~2.5MB, loaded only when first ```mermaid block detected
  • Rendered in main page DOM via ensureMermaid()
  • SVG baked into section HTML before iframe creation
  • No mermaid script in iframes

Each Section Iframe Contains

  • <head>
  •   <style> PDF_CSS + PAGED_CSS (inline text)
  •   <style> @page :first override (content sections)
  • <body>
  •   Pre-styled HTML (colors, numbers, underlines, mermaid SVG baked in)
  •   <script src> paged.polyfill.js (blob URL)
  •   <script> __sectionIndex / __sectionGen
  •   <script src> section-init.js (blob URL)

Marked Plugins (configured once in render.js)

  • renderer.heading — numbering, decorative underlines, --section-color CSS vars
  • renderer.code — mermaid blocks → placeholder divs (resolved async after parse)
  • hooks.postprocess — French typography (non-breaking spaces) + CSS variable injection on all block elements
  • _colorIdx — module-level state, set before each parse call (safe: marked.parse is sync)
